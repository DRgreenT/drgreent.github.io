HighlightIt

Paste text, optionally strip HTML, then highlight keywords (comma-separated or quoted). Everything updates live.

Input
Keywords e.g.: "black hole", LOFAR, 1420 MHz
Words: 0 Keywords: 0 Matches: 0
Tip: Phrases in "quotation marks" are treated as a single keyword. Special characters are handled safely (no regex issues).
Output (formatted + highlighted)
Output is plain text (markup is only used for highlighting). If “Strip HTML tags” is enabled, HTML input will be converted into clean text.
